# Bryan Pham — Portfolio

Personal portfolio site for Bryan Pham, a Data Science new grad (UC Riverside, 2026).

Built with **Next.js**, **TypeScript**, **Tailwind CSS**, **shadcn/ui**, and **Three.js** (animated dotted-surface hero).

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

## Live

Deployed on Vercel — pushes to `main` deploy automatically.
